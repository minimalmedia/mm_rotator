Rotator: An image rotator module for Drupal 7.x.

Usage:

Add a nodequeue at /content/nodequeue/add/nodequeue.
Create some Rotator Items at /node/add/rotator-item.
Add the Rotator Item nodes to the nodequeue.
Place one of the Rotator blocks in a block region.
